# HW-1 -Cooking App -
This project is about list of recipe and have a search feature by filter.

## Details:
1.  This design has been implemented in an app using Flutter.
2. The following are some of the widgets that were used:
   - **Image**
   - **AppBar**
   - **TextField**
   - **Row**
   - **Column**
   -  **Icon**
   - **Button**
   - **SingleChildScrollView**
   - **FilterChip**(as a way to filter content.)
3. The application provides navigation between pages.
